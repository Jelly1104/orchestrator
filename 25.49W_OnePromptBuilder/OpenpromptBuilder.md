# 기술 명세서

## 1. 아키텍처 개요

- 형태: 브라우저 기반 단일 페이지(SPA)
- 구조: 서버 없이 로컬 우선(Local-First)
- 실행 환경: 웹 브라우저 자바스크립트 엔진

---

## 2. 사용 기술

| 구분              | 기술                           | 비고                         |
| ----------------- | ------------------------------ | ---------------------------- |
| 화면 구성(프론트) | HTML5, 순수 JS 또는 React 제안 | 통합 시 Tailwind 조합 권장   |
| 스타일            | CSS3, Tailwind CSS             | 기존 그라디언트 스타일 유지  |
| 저장소            | 브라우저 LocalStorage          | JSON 문자열로 직렬화 후 저장 |
| AI 연결           | OpenAI API 사용                | fetch 기반 요청·응답 처리    |
| 실행 방식         | iframe 샌드박스                | `srcdoc`으로 코드 렌더링     |

---

## 3. 데이터 구조 (유사 코드 형식)

```ts
프로젝트 {
  id: 숫자
  이름: 문자
  설명: 문자
  생성일: 문자(날짜 형식)
  목표일: 문자(날짜 형식)
  단계별 진행: {
    mvp | alpha | beta | launch | growth 중 하나: {
      완료 여부: 참/거짓
      사용자 입력 프롬프트: 문자열 또는 없음
      산출물(선택): {
        기획서: 마크다운 형태
        코드: HTML/JS 전체 코드
      }
    }
  }
}
```
