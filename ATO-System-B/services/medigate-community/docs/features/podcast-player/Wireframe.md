## 3. 화면 설계

### S001: PodcastPlayer (메인 화면)

#### 레이아웃 (ASCII)

```
+--------------------------------------------------+
|                    HEADER                         |
|  +----------------------------------------------+ |
|  |  [Podcast Icon]  무찌마 일간 베스트 팟캐스트    | |
|  |  2026-01-08 생성                              | |
|  +----------------------------------------------+ |
+--------------------------------------------------+
|                                                  |
|                  PLAYER CONTROL                   |
|  +----------------------------------------------+ |
|  |                                              | |
|  |              [  ▶  PLAY  ]                   | |
|  |                                              | |
|  |        약 3분 | 450 단어                      | |
|  +----------------------------------------------+ |
|                                                  |
+--------------------------------------------------+
|                                                  |
|                 SCRIPT DISPLAY                    |
|  +----------------------------------------------+ |
|  |  [HOST]                                      | |
|  |  안녕하세요, 오늘의 무찌마 베스트를          | |
|  |  전해드리는 호스트입니다.                    | |
|  +----------------------------------------------+ |
|  |  [GUEST]                                     | |
|  |  네, 오늘도 핫한 게시물들이 많았네요.        | |
|  +----------------------------------------------+ |
|  |  [HOST]                                      | |
|  |  첫 번째 인기 게시물은...                    | |
|  +----------------------------------------------+ |
|  |                  ...                         | |
|  +----------------------------------------------+ |
|                                                  |
+--------------------------------------------------+
```

#### 상태별 레이아웃

**IDLE 상태 (초기)**
```
+----------------------------------------------+
|              [  ▶  PLAY  ]                   |
|        약 3분 | 450 단어                      |
+----------------------------------------------+
```

**PLAYING 상태 (재생 중)**
```
+----------------------------------------------+
|              [  ⏸  PAUSE  ]                  |
|        재생 중... 1:23 / 3:00                 |
+----------------------------------------------+
```

**PAUSED 상태 (일시정지)**
```
+----------------------------------------------+
|              [  ▶  RESUME  ]                 |
|        일시정지 1:23 / 3:00                   |
+----------------------------------------------+
```

#### 컴포넌트

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| Header | 공통 | 서비스 타이틀, 생성 일시 |
| PlayButton | 인터랙티브 | 재생/일시정지 토글 버튼 |
| MetaInfo | 정보 | 재생 시간, 단어 수 |
| ScriptDisplay | 목록 | 대본 대화 목록 |
| ScriptLine | 반복 | 개별 대화 항목 (HOST/GUEST) |

#### 인터랙션

| 액션 | 이벤트 | 결과 |
|------|--------|------|
| PlayButton 클릭 (idle) | onClick | playing 상태 전환, TTS 시뮬레이션 시작 |
| PlayButton 클릭 (playing) | onClick | paused 상태 전환 |
| PlayButton 클릭 (paused) | onClick | playing 상태 전환 (이어서 재생) |
| 키보드 Space | onKeyDown | 재생/일시정지 토글 |

### S002: ScriptDisplay (대본 표시)

#### 레이아웃 (ASCII)

```
+----------------------------------------------+
|  SCRIPT DISPLAY                              |
+----------------------------------------------+
|                                              |
|  +------------------------------------------+|
|  | [HOST Avatar]                            ||
|  | HOST                                     ||
|  | 안녕하세요, 오늘의 무찌마 베스트를        ||
|  | 전해드리는 호스트입니다.                  ||
|  +------------------------------------------+|
|                                              |
|  +------------------------------------------+|
|  |                            [GUEST Avatar]||
|  |                                    GUEST ||
|  |        네, 오늘도 핫한 게시물들이         ||
|  |        많았네요.                          ||
|  +------------------------------------------+|
|                                              |
+----------------------------------------------+
```

#### 컴포넌트

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| ScriptLine | 반복 | 개별 대화 라인 |
| SpeakerLabel | 텍스트 | HOST/GUEST 표시 |
| DialogText | 텍스트 | 대화 내용 |
| HighlightIndicator | 시각 | 현재 재생 중인 라인 강조 |
