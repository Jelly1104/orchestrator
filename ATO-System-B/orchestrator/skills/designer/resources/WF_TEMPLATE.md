## 2. 화면별 설계

### 2.1 [S01] 메인 화면

#### ASCII Wireframe

```
┌──────────────────────────────────────┐
│ [≡]  서비스명                    [⚙] │ ← 헤더
├──────────────────────────────────────┤
│                                      │
│  ┌────────────────────────────────┐  │
│  │ 메인 컨텐츠 영역                │  │
│  │                                │  │
│  │  ┌──────┐  ┌──────┐           │  │
│  │  │ Card │  │ Card │  ...      │  │
│  │  │  1   │  │  2   │           │  │
│  │  └──────┘  └──────┘           │  │
│  │                                │  │
│  └────────────────────────────────┘  │
│                                      │
├──────────────────────────────────────┤
│  [🏠]   [🔍]   [👤]   [⋮]          │ ← 탭바
└──────────────────────────────────────┘
```

#### 컴포넌트 설명

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 헤더 | AppBar | 서비스 타이틀, 메뉴, 설정 |
| 메인 | CardList | 카드 형태의 목록 |
| 탭바 | BottomNav | 주요 메뉴 네비게이션 |

#### 데이터 바인딩

| 요소 | 데이터 소스 | 필드 |
|------|------------|------|
| Card 제목 | API Response | `title` |
| Card 설명 | API Response | `description` |
| Card 이미지 | API Response | `thumbnailUrl` |

#### 인터랙션 정의

| 트리거 | 액션 | 결과 |
|--------|------|------|
| Card 탭 | Navigate | 상세 화면(S02) 이동 |
| 스크롤 끝 | Load More | 추가 데이터 로드 |
| 당겨서 새로고침 | Refresh | 목록 갱신 |

### 2.2 [S02] 상세 화면

#### ASCII Wireframe

```
┌──────────────────────────────────────┐
│ [←]  상세                        [⋮] │ ← 헤더
├──────────────────────────────────────┤
│                                      │
│  ┌────────────────────────────────┐  │
│  │                                │  │
│  │      히어로 이미지/콘텐츠       │  │
│  │                                │  │
│  └────────────────────────────────┘  │
│                                      │
│  제목 텍스트                         │
│  ──────────────────────────────────  │
│  서브 정보 | 태그1 | 태그2           │
│                                      │
│  ┌────────────────────────────────┐  │
│  │ 상세 설명                       │  │
│  │ ...                            │  │
│  └────────────────────────────────┘  │
│                                      │
├──────────────────────────────────────┤
│       [ 주요 액션 버튼 ]             │ ← CTA
└──────────────────────────────────────┘
```

#### 컴포넌트 설명

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 헤더 | AppBar | 뒤로가기, 더보기 메뉴 |
| 히어로 | ImageView | 대표 이미지 |
| 정보 | InfoSection | 제목, 태그, 메타 정보 |
| CTA | Button | 주요 액션 버튼 |

## 3. 공통 컴포넌트

### 3.1 Card Component

```
┌─────────────────────┐
│  [이미지]           │
│                     │
├─────────────────────┤
│  제목               │
│  설명 텍스트...     │
│  [태그] [태그]      │
└─────────────────────┘
```

**Props:**
- `title`: string
- `description`: string
- `imageUrl`: string
- `tags`: string[]
- `onPress`: function

### 3.2 Button Component

| 타입 | 스타일 | 용도 |
|------|--------|------|
| Primary | 채워진 배경 | 주요 액션 |
| Secondary | 테두리만 | 보조 액션 |
| Text | 텍스트만 | 링크성 액션 |

## 작성 가이드

### 필수 요소
1. 화면 목록 테이블
2. 화면별 ASCII Wireframe
3. 컴포넌트 설명
4. 데이터 바인딩
5. 인터랙션 정의

### ASCII 규칙
- `┌┐└┘├┤┬┴┼` 박스 문자 사용
- `│─` 라인 문자 사용
- `[텍스트]` 버튼/아이콘 표시
- `...` 생략 표시

### 품질 기준
- 모든 화면에 ID 부여 (S01, S02, ...)
- 컴포넌트는 재사용 가능하게 정의
- 인터랙션은 트리거-액션-결과 형식
