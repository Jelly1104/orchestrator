## 1. 추천 피드 메인 화면

```
┌─────────────────────────────────────────────────────────┐
│ [메디게이트 로고]    [검색]    [마이페이지] [알림]      │
├─────────────────────────────────────────────────────────┤
│ 채용정보 > 맞춤 추천                                    │
├─────────────────────────────────────────────────────────┤
│ ┌─ 탭 메뉴 ──────────────────────────────────────────┐  │
│ │ [전체공고] [맞춤추천*] [스크랩] [지원현황]          │  │
│ └─────────────────────────────────────────────────────┘  │
├─────────────────────────────────────────────────────────┤
│ 📍 김OO님을 위한 맞춤 채용공고 (5건)    [설정⚙️]       │
├─────────────────────────────────────────────────────────┤
│ ┌─ 추천 카드 #1 ────────────────────────────────────┐  │
│ │ 🏥 서울대학교병원 내과 봉직의 모집                │  │
│ │ 💰 연봉 1억2천만원    📍 서울 종로구              │  │
│ │ 📊 매칭점수: 87점 ⭐⭐⭐⭐⭐                      │  │
│ │ ✅ 전문과목 일치 + 근무형태 일치 + 지역 선호       │  │
│ │ [상세보기] [지원하기] [❤️ 스크랩]                   │  │

│                                                         │

│ │ 💰 일급 50만원       📍 서울 강남구              │  │
│ │ 📊 매칭점수: 76점 ⭐⭐⭐⭐                        │  │
│ │ ✅ 전문과목 일치 + 신규공고                       │  │
│ │ [상세보기] [지원하기] [❤️ 스크랩]                   │  │
│ └─────────────────────────────────────────────────────┘  │
│                                                         │
│ ... (3건 더) ...                                       │
├─────────────────────────────────────────────────────────┤
│ [더 많은 공고 보기]                                     │
└─────────────────────────────────────────────────────────┘
```

### 컴포넌트 매핑
- **추천 카드**: `CBIZ_RECJOB.TITLE`, `CBIZ_RECJOB.SALARY_MIN`, `CODE_LOC.SIDO`
- **매칭 점수**: 추천 알고리즘 계산 결과 (0-100)
- **추천 이유**: 전문과목/근무형태/지역 일치 여부

## 2. 추천 상세 화면 ("왜 추천되었나요?")

```
┌─────────────────────────────────────────────────────────┐
│ ← 뒤로가기   추천 상세 분석                              │
├─────────────────────────────────────────────────────────┤
│ 🏥 서울대학교병원 내과 봉직의 모집                      │
│ 📊 종합 매칭점수: 87점 ⭐⭐⭐⭐⭐                      │
├─────────────────────────────────────────────────────────┤
│ ┌─ 매칭 분석 ──────────────────────────────────────┐  │
│ │ ✅ 전문과목 매칭 (40점/40점)                      │  │
│ │    귀하: 내과 전문의 ↔ 공고: 내과 봉직의 모집      │  │
│ │                                                   │  │
│ │ ✅ 근무형태 매칭 (26점/30점)                      │  │
│ │    귀하: 봉직의 선호 ↔ 공고: 정규직 봉직의        │  │
│ │                                                   │  │
│ │ ⚠️ 지역 매칭 (14점/20점)                         │  │
│ │    귀하: 강남구 선호 ↔ 공고: 종로구 (인접)        │  │
│ │                                                   │  │
│ │ ✅ 신규공고 보너스 (7점/10점)                     │  │
│ │    3일 전 등록된 신규 공고                        │  │
│ └─────────────────────────────────────────────────────┘  │
├─────────────────────────────────────────────────────────┤
│ 💡 추천 개선을 위한 피드백                               │
│ ┌─────────────────────────────────────────────────────┐  │
│ │ 이 추천이 도움이 되셨나요?                          │  │
│ │ [👍 도움됨] [👎 도움안됨] [💬 의견남기기]            │  │
│ └─────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

### 데이터 매핑
- **전문과목 매칭**: `USER_DETAIL.U_MAJOR_CODE_1` ↔ `CBIZ_RECJOB.MAJOR_CODE`
- **근무형태 매칭**: `USER_DETAIL.U_WORK_TYPE_1` ↔ `CBIZ_RECJOB.INVITE_TYPE`
- **지역 매칭**: `USER_DETAIL.U_OFFICE_ZIP` vs `CBIZ_RECJOB.WORK_AREA_CODE`

## 3. 추천 설정 화면

```
┌─────────────────────────────────────────────────────────┐
│ ← 뒤로가기   맞춤 추천 설정                              │
├─────────────────────────────────────────────────────────┤
│ ┌─ 추천 조건 ──────────────────────────────────────┐  │
│ │ 📋 전문과목 (필수)                                │  │
│ │ [내과 ✅] [외과 ☐] [가정의학과 ☐] [기타 ☐]        │  │
│ │                                                   │  │
│ │ 💼 선호 근무형태                                  │  │
│ │ [봉직의 ✅] [대진의 ☐] [전공의 ☐] [기타 ☐]        │  │
│ │                                                   │  │
│ │ 📍 선호 지역 (최대 3개)                           │  │
│ │ [서울 강남구 ✅] [서울 종로구 ☐]                  │  │
│ │ [경기 성남시 ☐] [+ 지역추가]                      │  │
│ │                                                   │  │
│ │ 💰 희망 연봉 범위                                 │  │
│ │ 최소: [8,000만원 ▼] 최대: [1억5천만원 ▼]         │  │
│ └─────────────────────────────────────────────────────┘  │
├─────────────────────────────────────────────────────────┤
│ ┌─ 알림 설정 ──────────────────────────────────────┐  │
│ │ 🔔 신규 매칭 공고 알림                            │  │
│ │ [☐] 푸시 알림   [☐] 이메일 알림                   │  │
│ │                                                   │  │
│ │ ⏰ 알림 주기                                      │  │
│ │ (●) 즉시   ( ) 하루 1회   ( ) 주 1회             │  │
│ └─────────────────────────────────────────────────────┘  │
├─────────────────────────────────────────────────────────┤
│                     [저장하기]                          │
└─────────────────────────────────────────────────────────┘
```

### 컴포넌트 데이터 소스
- **전문과목**: `CODE_MASTER` WHERE `CODE_TYPE='MAJOR'`
- **근무형태**: `CODE_MASTER` WHERE `CODE_TYPE='WORK_TYPE'`  
- **지역**: `CODE_LOC` 테이블 (시/군/구 레벨)
