#### 컴포넌트

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| Header | 공통 | 제목 + Refresh 버튼 |
| SkillCard | 반복 x7 | 개별 Skill 정보 카드 |
| StatusIndicator | 하위 | 상태 dot (초록/회색) |
| Footer | 공통 | 통계 요약 바 |

#### 컴포넌트 상세

##### Header

```
+--------------------------------------------------+
|  [Wrench Icon] Skill Dashboard         [Refresh] |
+--------------------------------------------------+

- 왼쪽: 아이콘 + 제목 텍스트
- 오른쪽: Refresh 버튼
- 배경: 흰색, 하단 border
```

##### SkillCard

```
+------------------+
|  skill-name      |   <- bold, 16px
|  v1.0.0          |   <- gray, 14px
|  [*] active      |   <- 상태 dot + 텍스트
+------------------+

- 배경: 흰색
- 그림자: shadow-md
- 패딩: p-4
- 라운드: rounded-lg
```

##### StatusIndicator

```
Active:   [*] active    <- 초록색 dot (#22c55e)
Inactive: [o] inactive  <- 회색 dot (#9ca3af)
```

##### Footer

```
+--------------------------------------------------+
|  Total: 7 Skills  |  Active: 6  |  Inactive: 1   |
+--------------------------------------------------+

- 배경: gray-100
- 텍스트: 중앙 정렬
- 구분선: | (파이프)
```

#### 인터랙션

| 액션 | 이벤트 | 결과 |
|------|--------|------|
| Refresh 버튼 클릭 | onClick | 데이터 갱신 (현재는 정적) |
| Card 호버 | onMouseEnter | 그림자 강조 (shadow-lg) |
