## 5. 상호작용 정의

### 5.1 실시간 업데이트
- WebSocket 연결을 통한 실시간 데이터 업데이트
- 5초마다 상태 정보 갱신
- 중요 이벤트 발생시 즉시 알림

### 5.2 사용자 액션
| 액션 | 트리거 | 결과 |
|------|--------|------|
| 테스트 실행 | 버튼 클릭 | 비동기 테스트 시작, 진행률 표시 |
| 에이전트 재시작 | 컨텍스트 메뉴 | 해당 에이전트 재부팅 |
| 로그 필터링 | 검색/필터 | 조건에 맞는 로그만 표시 |
| 알림 설정 | 설정 페이지 | 알림 규칙 저장 |
