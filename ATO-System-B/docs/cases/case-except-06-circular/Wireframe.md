# Wireframe.md - 화면 설계

## A 기능 화면
- 입력 필드: B의 결과
- 전송 버튼: C로 결과 전송

## B 기능 화면
- 입력 필드: C의 결과
- 전송 버튼: A로 결과 전송

## C 기능 화면
- 입력 필드: A의 결과
- 전송 버튼: B로 결과 전송

## 공통 컴포넌트
- 결과 표시 영역: 각 기능의 결과를 각각의 입력 필드 하단에 표시
- 의존성 경고 메시지: 필요한 선행 작업 완료 필요시 다음 동작을 제한

### 데이터 매핑
- B의 결과 데이터 필드 → A 입력 필드
- C의 결과 데이터 필드 → B 입력 필드
- A의 결과 데이터 필드 → C 입력 필드
