### 3. 컴포넌트 설명
- **글로벌 네비게이션 바 (GNB)**: 주요 페이지로의 빠른 이동 제공.
- **로컬 네비게이션 바 (LNB)**: 현재 페이지 관련 세부 항목 탐색 지원.
- **Skills 카드**: Skill의 현재 상태와 버전을 요약하여 제공.
- **상태 뱃지**: Skill의 활성/비활성 상태를 나타냄.
- **바디 컨텐츠**: 각각의 페이지에 적합한 정보를 표시.

### 4. 인터랙션
- **메뉴 클릭**: 페이지 이동.
- **Skill 카드 클릭**: /skills/list/details/:id 페이지로 이동.
- **설정 옵션 클릭**: 설정 변경 가능.

### 5. 데이터 바인딩
- **각 페이지**는 특정 API로부터 데이터를 가져와 컴포넌트에 바인딩.
- **Skills 목록 API**: /skills/list/overview의 카드 데이터 제공.
- **Skill 상세 정보 API**: /skills/list/details/:id에 상세 정보 제공.
- **Skill 카테고리 목록 API**: /skills/categories에 카테고리 데이터 제공.
- **사용자 설정 및 Preferences API**: /skills/settings의 설정 데이터 제공.
