#### 컴포넌트 구조

| 컴포넌트 | 타입 | 설명 | TailwindCSS |
|----------|------|------|-------------|
| Header | 공통 | 서비스 제목 + 날짜 | `bg-gradient-to-r from-blue-600 to-indigo-700` |
| ScriptDisplay | 핵심 | 대본 표시 (스크롤) | `bg-white rounded-lg shadow-md overflow-y-auto` |
| ScriptLine | 반복 | 개별 대사 | `p-3 rounded` + Host/Guest별 색상 |
| PlayButton | 컨트롤 | 재생/일시정지 버튼 | `bg-blue-500 hover:bg-blue-600 text-white` |
| ProgressBar | 컨트롤 | 진행률 표시 | `bg-gray-200` + `bg-blue-500` |
| BestPostList | 목록 | 베스트 게시물 요약 | `bg-gray-50 rounded-lg` |
| BestPostItem | 반복 | 개별 게시물 항목 | `flex justify-between` |

#### 인터랙션

| 액션 | 이벤트 | 결과 |
|------|--------|------|
| PlayButton 클릭 | onClick | PLAYING ↔ PAUSED 상태 토글 |
| 대본 자동 스크롤 | 재생 중 | 현재 대사 위치로 자동 스크롤 |
| 대본 수동 스크롤 | onScroll | 자동 스크롤 일시 중단 |

## 3. 컴포넌트별 상세 설계

### 3.1 ScriptDisplay

```
+------------------------------------------------+
| [Host - Dr. AI]                    bg-blue-50   |
| 안녕하세요, 무찌마 일간 베스트 팟캐스트입니다.   |
| 오늘의 화제 글을 살펴보겠습니다.                 |
+------------------------------------------------+
| [Guest]                           bg-green-50   |
| 네, 오늘 가장 뜨거운 글은 의료분쟁              |
| 경험담이네요. 무려 추천 736개를 받았습니다.      |
+------------------------------------------------+
| [Host - Dr. AI]                    bg-blue-50   |
| 의료분쟁은 정말 힘든 상황이죠...                |
+------------------------------------------------+
```

### 3.2 PlayButton 상태별 표시

| 상태 | 버튼 텍스트 | 아이콘 | 스타일 |
|------|-------------|--------|--------|
| LOADING | 로딩 중... | 스피너 | `bg-gray-400 cursor-wait` |
| READY | ▶ 재생 | Play | `bg-blue-500 hover:bg-blue-600` |
| PLAYING | ⏸ 일시정지 | Pause | `bg-orange-500 hover:bg-orange-600` |
| PAUSED | ▶ 계속 재생 | Play | `bg-blue-500 hover:bg-blue-600` |

## 4. 반응형 고려사항

| 브레이크포인트 | 레이아웃 | 변경 사항 |
|----------------|----------|-----------|
| Mobile (< 640px) | 1열 세로 | 전체 너비, 작은 패딩 |
| Tablet (640px~1024px) | 1열 세로 | 중간 패딩, 최대 너비 제한 |
| Desktop (> 1024px) | 1열 세로 | max-w-2xl 중앙 정렬 |

### 반응형 클래스

```
Container: max-w-2xl mx-auto px-4 sm:px-6 lg:px-8
ScriptDisplay: h-64 sm:h-80 lg:h-96
PlayButton: w-full sm:w-auto px-8 py-4
```
