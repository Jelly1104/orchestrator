## 3. 인사이트

### 주요 발견

1. **의료분쟁 관련 콘텐츠 압도적 인기**
   - 상위 5건 중 3건이 의료분쟁 경험담
   - 1위 게시물 인기점수(7,360)가 2위(2,850)의 2.5배 이상

2. **에세이/감성 콘텐츠 반응 좋음**
   - "진료실 에세이", "잠잘곳 있고 출근할곳 있고" 등 감성 글 인기

3. **추천수 중심 인기도**
   - 모든 게시물의 READ_CNT가 0 (조회수 미집계 또는 별도 관리)
   - 추천수(AGREE_CNT)가 실질적인 인기 지표

### 콘텐츠 특성

| 카테고리 | 비율 | 설명 |
|----------|------|------|
| 의료분쟁 | 60% | 분쟁 경험, 대응 방법 |
| 감성/에세이 | 40% | 일상 공감, 철학적 성찰 |
