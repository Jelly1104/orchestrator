## 3. 인사이트

### 3.1 주요 패턴

| 패턴 | 설명 |
|------|------|
| **의료 현장 스트레스** | 의료분쟁, 환자 응대 스트레스 관련 글이 높은 관심 |
| **의사 라이프스타일** | 결혼, 취미, 재테크 등 일상 관련 글도 인기 |
| **커뮤니티 정서 공유** | 고충 토로, 공감대 형성 글에 댓글 참여 활발 |

### 3.2 Engagement 분석

```
평균 Engagement Score: 280
평균 댓글 수: 85.3
댓글 기여도: (댓글수 * 3) / Engagement Score ≈ 90%
```

- 댓글 참여가 Engagement의 핵심 지표
- 조회수(READ_CNT)가 0으로 기록된 것은 DB 집계 방식의 특성으로 추정

### 3.3 콘텐츠 카테고리 분류

| 카테고리 | 게시물 수 | 대표 주제 |
|----------|-----------|-----------|
| 의료 현장 | 3건 | 의료분쟁, 환자 응대, 오진 사례 |
| 일상/라이프 | 4건 | 결혼, 취미, 인간관계 |
| 시사/이슈 | 2건 | 사회 이슈 코멘트 |
| 재테크 | 1건 | 부동산, 자산 |
