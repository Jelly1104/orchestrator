#### 컴포넌트

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| Header | 고정 | 팟캐스트 제목, 에피소드 번호 |
| SectionBlock | 반복 | 섹션별 대본 컨테이너 |
| SpeakerLine | 반복 | 발화자별 대사 라인 |
| TimeCode | 메타 | 섹션별 예상 시간 표시 |

#### 인터랙션

| 액션 | 이벤트 | 결과 |
|------|--------|------|
| 섹션 클릭 | onClick | 해당 섹션 확대/축소 토글 |
| 발화자 클릭 | onClick | 해당 발화자 대사만 하이라이트 |
| 복사 버튼 | onClick | 대본 전체 클립보드 복사 |

### S002: 발화자 구분 형식

#### 발화자 스타일 가이드

```
+------------------+------------------+
|     🎤 Host      |     🎧 Guest     |
+------------------+------------------+
| 배경: #E3F2FD    | 배경: #E8F5E9    |
| 텍스트: #1565C0  | 텍스트: #2E7D32  |
| 아이콘: 🎤       | 아이콘: 🎧       |
| 역할: 진행/질문  | 역할: 분석/답변  |
+------------------+------------------+
```

#### 대사 포맷 규칙

```
[발화자 표기] "대사 내용"

예시:
🎤 Host: "안녕하세요, 오늘의 무찌마 핫토픽입니다!"
🎧 Guest: "안녕하세요! 오늘도 재미있는 이야기 나눠볼게요."
```
