## 2. 컴포넌트 데이터 매핑 (알림 목록)

| Zone | 컴포넌트 | 데이터 소스 | 비고 |
|------|---------|------------|------|
| Header | `PageTitle` | "알림" | |
| Header.Badge | `Badge` | `COUNT(READ_FLAG='N')` | 안읽은 개수 |
| List | `NotificationItem` | `NOTIFICATION` | 반복 렌더링 |
| Item.Icon | `Icon` | `NOTI_TYPE` | NOTICE=📢, EVENT=🎉, SYSTEM=⚙️ |
| Item.Title | `Text` | `TITLE` | |
| Item.Message | `Text` | `MESSAGE` | 2줄 truncate |
| Item.Time | `Text` | `REG_DATE` | 상대 시간 표시 |
| Item.Dot | `Dot` | `READ_FLAG` | 'N'이면 표시 |

## 4. 컴포넌트 데이터 매핑 (발송 폼)

| Zone | 컴포넌트 | 데이터 | 비고 |
|------|---------|--------|------|
| Target | `RadioGroup` | 'ALL' / 'SPECIFIC' | |
| Target.Input | `TextInput` | U_ID 목록 | SPECIFIC 선택 시만 |
| Type | `Select` | NOTICE/EVENT/SYSTEM | |
| Title | `TextInput` | TITLE | max 100자 |
| Message | `Textarea` | MESSAGE | max 500자 |
| Submit | `Button` | - | POST 요청 |
