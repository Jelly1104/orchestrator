# PRD: 활성 사용자 세그먼트 분석

| **Case ID** | success-a-analysis |
|-------------|-------------------|
| **Pipeline** | analysis |

## 1. 목적
**의사 회원 활성도 분석**

최근 30일 기준 의사 회원의 활성도를 분석하여 세그먼트별 현황을 파악합니다.

## 2. 타겟 유저
- 마케팅팀 (세그먼트별 캠페인 기획)
- 운영팀 (휴면 회원 관리)

## 3. 핵심 기능

### 3.1 데이터 분석 요구사항
- [ ] 의사 회원(U_KIND='DOC001') 중 활성(U_ALIVE='Y') 회원 수 조회
- [ ] 최근 30일 로그인 기록 기준 활성/비활성 분류
- [ ] 전문과목(U_MAJOR_CODE_1)별 분포 분석

### 3.2 분석 결과 산출물
- [ ] 전체 의사 회원 수
- [ ] 활성 회원 비율
- [ ] 전문과목별 Top 5

## 4. 성공 지표
- SQL 쿼리 정상 실행
- 분석 리포트 생성
