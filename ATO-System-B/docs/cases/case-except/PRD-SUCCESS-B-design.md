# PRD: 마이페이지 프로필 편집 화면

| **Case ID** | success-b-design |
|-------------|-----------------|
| **Pipeline** | design |

## 1. 목적
**프로필 편집 UX 개선**

회원이 자신의 프로필 정보를 쉽게 수정할 수 있는 화면을 설계합니다.

## 2. 타겟 유저
- 의사 회원 (프로필 정보 수정 필요)
- 약사 회원

## 3. 핵심 기능

### 3.1 프로필 편집 화면
- [ ] 기본 정보 표시 (이름, 이메일)
- [ ] 전문과목 선택 드롭다운
- [ ] 근무지 정보 입력 폼
- [ ] 프로필 사진 업로드

### 3.2 저장 및 취소
- [ ] 저장 버튼 클릭 시 유효성 검사
- [ ] 취소 버튼 클릭 시 이전 화면 복귀

## 4. 성공 지표
- IA 문서 생성
- Wireframe 문서 생성
- SDD 문서 생성
