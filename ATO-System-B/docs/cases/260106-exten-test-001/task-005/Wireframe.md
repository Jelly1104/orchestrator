### 3. 컴포넌트 설명

- **Global Navigation Bar (GNB)**: 모든 페이지 상단에 위치하며 주요 항목인 Dashboard와 Settings를 포함합니다.
- **Local Navigation Bar (LNB)**: Skills 목록 페이지에서 사용되며, Skills Overview와 Add New Skill을 선택할 수 있습니다.
- **Tabs**: Skill 세부 정보 페이지에서 사용되며, Overview, Edit Skill, Version History 정보를 선택할 수 있습니다.
- **Skill List**: Skills 목록 페이지에 표시되는 컴포넌트로 Skill 이름, 상태 뱃지, 버전 정보를 포함합니다.

---

### 4. 인터랙션

- **GNB 클릭**: Dashboard와 Settings를 클릭하면 해당 페이지로 이동합니다.
- **LNB 클릭**: Skills Overview와 Add New Skill을 클릭하여 각각의 페이지로 이동합니다.
- **탭 클릭**: Skill 세부 정보 페이지에서 탭을 클릭하면 해당 탭의 콘텐츠가 표시됩니다.
- **Skill 클릭**: Skills 목록 페이지에서 Skill을 클릭하면 Skill 세부 정보 페이지로 이동합니다.
- **저장 버튼 클릭**: Skill 편집 페이지에서 저장 버튼을 클릭하면 변경 사항이 저장됩니다.

---

### 5. 데이터 바인딩

- **Skills 목록 페이지**: Skills API에서 Skill 목록 데이터를 받아와 목록을 구성합니다.
- **Skill 세부 정보 페이지**: Skill Detail API에서 세부 정보를 받아와 페이지를 구성합니다.
- **Skill 편집 페이지**: Skill Update API에서 정보를 받아와 작성 및 수정할 수 있습니다.
