| 상태 뱃지 | Skill | `status` |
| 설명 | Skill | `description` |

#### 인터랙션 정의

| 트리거 | 액션 | 결과 |
|--------|------|------|
| 카드 Hover | Highlight | 카드 배경색 변경, 그림자 추가 |
| 뷰포트 리사이즈 | Reflow | 그리드 열 수 조정 (1~3열) |

## 3. 공통 컴포넌트

### 3.1 SkillCard Component

```
┌───────────────────────────┐
│  [Skill Name]    ● active │ ← CardHeader
├───────────────────────────┤
│                           │
│       v1.3.0              │ ← CardBody (Version)
│                           │
├───────────────────────────┤
│  SQL 쿼리 생성 및         │ ← CardFooter (Description)
│  데이터 분석              │
└───────────────────────────┘
```

**Props:**

| Prop | Type | Required | 설명 |
|------|------|----------|------|
| `name` | string | Yes | Skill 이름 |
| `version` | string | Yes | 버전 (v{x}.{y}.{z}) |
| `status` | "active" \| "inactive" | Yes | 상태 |
| `description` | string | Yes | 설명 텍스트 |

### 3.2 StatusBadge Component

```
┌───────────┐
│ ● active  │  (녹색 배경)
└───────────┘

┌───────────┐
│ ● inactive│  (회색 배경)
└───────────┘
```

**Props:**

| Prop | Type | Required | 설명 |
|------|------|----------|------|
| `status` | "active" \| "inactive" | Yes | 상태 값 |

**스타일 매핑:**

| Status | Badge 색상 | 텍스트 색상 |
|--------|------------|-------------|
| active | `bg-green-100` | `text-green-800` |
| inactive | `bg-gray-100` | `text-gray-500` |

## 5. 스타일 가이드

### 5.1 색상 팔레트

| 용도 | TailwindCSS | Hex |
|------|-------------|-----|
| 배경 | `bg-gray-50` | #F9FAFB |
| 카드 배경 | `bg-white` | #FFFFFF |
| 카드 테두리 | `border-gray-200` | #E5E7EB |
| 제목 텍스트 | `text-gray-900` | #111827 |
| 본문 텍스트 | `text-gray-600` | #4B5563 |
| Active 뱃지 | `bg-green-100 text-green-800` | - |
| Inactive 뱃지 | `bg-gray-100 text-gray-500` | - |

### 5.2 타이포그래피

| 요소 | TailwindCSS | 용도 |
|------|-------------|------|
| 페이지 타이틀 | `text-2xl font-bold` | Skills Dashboard |
| 카드 타이틀 | `text-lg font-semibold` | Skill 이름 |
| 버전 | `text-xl font-mono` | v1.3.0 |
| 설명 | `text-sm text-gray-600` | Description |
| 뱃지 | `text-xs font-medium` | active/inactive |

### 5.3 간격 (Spacing)

| 요소 | TailwindCSS | 값 |
|------|-------------|-----|
| 페이지 패딩 | `p-6` | 24px |
| 그리드 갭 | `gap-6` | 24px |
| 카드 패딩 | `p-4` | 16px |
| 카드 내부 간격 | `space-y-2` | 8px |
