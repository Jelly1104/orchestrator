# Wireframe.md - 화면 설계

## 1. 회원 시스템
### 1.1 회원가입 플로우
- 구성 요소: 입력 필드 (이메일, 비밀번호, 프로필 정보)
- 데이터 매핑: USERS 테이블 사용
- 상태 관리: Redux 또는 Context API 활용

## 2. 커뮤니티 시스템
### 2.1 게시판
- 구성 요소: 게시글 리스트, 검색/필터 기능, 페이지네이션
- 데이터 매핑: BOARD_MUZZIMA 테이블 사용
- 상태 관리: SWR로 데이터 페칭

## 3. 채용 시스템
### 3.1 채용공고 등록
- 구성 요소: 입력 폼, 미리보기, 저장 버튼
- 데이터 매핑: CBIZ_RECJOB 테이블 사용

## 4. 임대/양도 시스템
### 4.1 매물 등록
- 구성 요소: 입력 필드 (주소, 가격, 설명), 이미지 업로드
- 데이터 매핑: CBIZ_LEASE 테이블 사용
- 상태 관리: Formik 라이브러리 활용
