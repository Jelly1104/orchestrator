# Wireframe.md - 화면 설계

## 대시보드 레이아웃

```
┌────────────────────────────────────────────┐
│  Skills Dashboard                          │
├────────────────────────────────────────────┤
│                                            │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐       │
│  │ Query   │ │ Profiler│ │ Designer│       │
│  │ v1.3.0  │ │ v1.3.0  │ │ v2.3.0  │       │
│  │ ●active │ │ ●active │ │ ●active │       │
│  └─────────┘ └─────────┘ └─────────┘       │
│                                            │
│  ┌─────────┐ ┌─────────┐                   │
│  │ Coder   │ │ Reviewer│                   │
│  │ v1.4.0  │ │ v1.3.0  │                   │
│  │ ●active │ │ ●active │                   │
│  └─────────┘ └─────────┘                   │
│                                            │
└────────────────────────────────────────────┘
```

### 컴포넌트 세부 사항

- **SkillCard**
  - **내용**: Skill 이름, 버전, 활성 상태
  - **시각 요소**: 활성 상태에 따라 뱃지 색상 변경 (active: 녹색, inactive: 회색)
  - **스타일링 도구**: TailwindCSS 활용
