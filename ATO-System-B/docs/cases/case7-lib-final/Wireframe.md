## 2. 컴포넌트 구성

### Header Component
- 제목: "활성 사용자 현황 대시보드"
- 새로고침 버튼
- 마지막 업데이트 시간 표시

### Filter Component
- 기간 선택: DateRangePicker
- 사용자 유형 선택: Dropdown
- 적용/리셋 버튼

### KPI Cards Component
```javascript
const kpiCards = [
  { title: "활성 사용자", value: "0", unit: "명", trend: "0%" },
  { title: "전체 사용자", value: "203,184", unit: "명", trend: "+2.1%" },
  { title: "월간 신규", value: "408", unit: "명", trend: "+5.3%" },
  { title: "활성화율", value: "0.0", unit: "%", trend: "-0.1%" }
]
```

### Chart Components
1. **유형별 분포 파이 차트**
   - 라이브러리: Chart.js / Recharts
   - 데이터: USER_KIND 기반 집계

2. **월별 추이 라인 차트**
   - X축: 월 (2024-01 ~ 2025-12)
   - Y축: 신규 가입자 수
   - 트렌드 라인 포함

### Data Table Component
- 페이지네이션: 10개/페이지
- 정렬 기능: 사용자수, 비율 기준
- 내보내기 기능: CSV/Excel

## 3. 반응형 레이아웃

### Desktop (≥1200px)
- 4열 KPI 카드
- 차트 2개 나란히 배치
- 전체 테이블 표시

### Tablet (768px-1199px)
- 2열 KPI 카드
- 차트 세로 배치
- 테이블 가로 스크롤

### Mobile (≤767px)
- 1열 KPI 카드
- 차트 세로 배치
- 테이블 축약 버전

## 4. 데이터 매핑

| UI 요소 | 데이터 소스 | 컬럼 |
|---------|------------|------|
| 활성 사용자 수 | USERS | COUNT(*) WHERE U_ALIVE='Y' |
| 전체 사용자 수 | USERS | COUNT(*) |
| 유형별 분포 | USERS + CODE_MASTER | U_KIND, CODE_NAME |
| 월별 추이 | USERS | U_REG_DATE |
| 전문과목 분포 | USER_DETAIL + CODE_MASTER | U_MAJOR_CODE_1 |
