# Wireframe.md - 화면 설계

## 1. 무찌마 일간 베스트 팟캐스트 페이지 (/podcast/daily-best)

### 상단 내비게이션
- 로고
- 홈 링크
- 팟캐스트 링크
- 로그인/로그아웃 버튼

### 콘텐츠 섹션
- 좌측: 게시물 목록
  - Item: 제목, 조회수+동의수 합계, 피드백 버튼
- 중앙: 대본 생성 및 메타데이터
  - 대본 표시 창 (Host/Guest 구분 강조)
  - 메타데이터 (발화 속도, 감정 태그 등)
- 우측: TTS 플레이어
  - 재생/일시정지/정지 버튼
  - 음색 조정 슬라이더 (Host/Guest)

### 하단
- 저작권 정보
- 서비스 문의 링크
