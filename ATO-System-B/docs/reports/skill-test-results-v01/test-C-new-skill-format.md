## 4. 인터랙션 정의

### 4.1 필터 패널

| 액션 | 동작 |
|------|------|
| 드롭다운 선택 | 해당 필터 옵션 표시 |
| [적용하기] 클릭 | 모든 차트/테이블 데이터 갱신 |
| [초기화] 클릭 | 모든 필터 기본값으로 리셋 |

### 4.2 데이터 테이블

| 액션 | 동작 |
|------|------|
| 컬럼 헤더 클릭 | 해당 컬럼 기준 정렬 (ASC/DESC 토글) |
| 검색어 입력 | 실시간 필터링 (300ms debounce) |
| 페이지 버튼 클릭 | 해당 페이지 데이터 로드 |
| [CSV 다운로드] | 현재 필터 기준 데이터 다운로드 |

### 4.3 차트

| 액션 | 동작 |
|------|------|
| 차트 영역 호버 | 툴팁으로 상세 수치 표시 |
| 범례 클릭 | 해당 시리즈 토글 (표시/숨김) |
| [전체보기] 클릭 | 상세 모달 또는 페이지 이동 |
