## 5. 반응형 컴포넌트 정의

### 5.1 대시보드 카드 컴포넌트
```
Desktop (≥1024px): 3열 그리드
Tablet (768-1023px): 2열 그리드  
Mobile (≤767px): 1열 스택
```

### 5.2 차트 컴포넌트
```
- ProgressBar: 파이프라인 진행률
- DonutChart: 세그먼트 분포
- LineChart: 로그인 트렌드
- BarChart: 전문과목별 분포
```

### 5.3 필터 패널
```
Desktop: 사이드바 형태
Mobile: 드롭다운 형태
- Phase 선택 (A/B/C)
- 상태 필터 (성공/실패/진행중)
- 시간 범위 (1시간/6시간/24시간)
```
