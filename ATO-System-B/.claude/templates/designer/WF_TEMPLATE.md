## 3. 화면 설계

### S001: {화면명}

#### 레이아웃 (ASCII)

```
+------------------------------------------+
|              Header                       |
+------------------------------------------+
|                                          |
|            Main Content                  |
|                                          |
|   +--------+  +--------+  +--------+     |
|   | Card 1 |  | Card 2 |  | Card 3 |     |
|   +--------+  +--------+  +--------+     |
|                                          |
+------------------------------------------+
|              Footer                       |
+------------------------------------------+
```

#### 컴포넌트

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| Header | 공통 | 페이지 헤더 |
| Card | 반복 | 데이터 카드 |
| Footer | 공통 | 페이지 푸터 |

#### 인터랙션

| 액션 | 이벤트 | 결과 |
|------|--------|------|
| Card 클릭 | onClick | 상세 페이지로 이동 |
