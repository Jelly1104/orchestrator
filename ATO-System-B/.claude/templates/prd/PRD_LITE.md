### PRD_LITE vs PRD_FULL 선택 기준

| 상황 | 권장 템플릿 |

| 빠른 프로토타이핑, MVP 검증 | PRD_LITE (Fast-track) |
| 리스크가 높은 신규 기능 | **PRD_FULL (Deep-dive)** |
| 시장/경쟁 분석이 필요한 경우 | **PRD_FULL (Deep-dive)** |
