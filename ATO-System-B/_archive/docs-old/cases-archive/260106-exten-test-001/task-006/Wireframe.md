## 컴포넌트 설명
- **Skills Dashboard**
  - 최상단 타이틀
  - 하단에 각 `SkillCard`가 2행 3열 레이아웃

- **SkillCard**
  - 상단: Skill 이름
  - 중간: 버전 정보
  - 하단: 상태 뱃지 (●active)

## 데이터 매핑
- `SkillCard.tsx`는 props로 `Skill name`, `version`, `status`를 받아 해당 내용을 카드에 표시
