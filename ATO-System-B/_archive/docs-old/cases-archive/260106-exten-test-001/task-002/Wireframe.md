# Wireframe 문서

## 1. 화면 목록
1. 대시보드 메인 (`/dashboard`)
2. 스킬 대시보드 (`/dashboard/skills`)
3. 스킬 개요 (`/dashboard/skills/overview`)
4. 스킬 상세 정보 (`/dashboard/skills/:id`)
5. 스킬 설정 (`/dashboard/skills/settings`)
6. 리포트 (`/dashboard/reports`)

## 3. 컴포넌트 설명

- **GNB (Global Navigation Bar)**: 모든 페이지에서 상단에 고정되어 이동할 수 있는 전역 네비게이션 바. 각 버튼은 대시보드, 스킬 대시보드, 리포트로 이동.
- **카드**: 스킬 대시보드에서 스킬을 카드 형태로 표시함. 각 카드는 ID와 함께 요약 정보를 제공.
- **탭 (스킬 페이지 내)**: 스킬 내비게이션 탭에서 사용자가 개요, 상세 정보, 설정 화면 간을 손쉽게 전환 가능.

## 4. 인터랙션

- **카드 클릭**: 스킬 카드 클릭 시 해당 스킬의 상세 페이지로 이동.
- **탭 클릭**: 스킬 상세 페이지 내 탭 클릭 시 탭에 해당하는 페이지 내용으로 변경 및 렌더링.
- **상태 및 버전 정보**: 스킬 페이지 내 선택한 스킬의 상태와 버전 표시.

## 5. 데이터 바인딩

- **대시보드 메인**: 최근 활동 정보는 실시간으로 데이터 소스로부터 가져와 표시.
- **스킬 대시보드**: Skills API를 통해 스킬 목록 및 상태, 버전 정보를 실시간으로 가져와 연결.
- **스킬 상세**: Skill Detail API로부터 각 스킬의 상세한 정보를 받아와 바인딩하여 표시.
