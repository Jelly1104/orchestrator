## 2. 반응형 레이아웃

### 2.1 Desktop (>= 1024px)
- Key Metrics: 4열 가로 배치
- Chart: 전체 너비

### 2.2 Tablet (768px ~ 1023px)
- Key Metrics: 2x2 그리드
- Chart: 전체 너비

### 2.3 Mobile (< 768px)
- Key Metrics: 1x4 세로 스택
- Chart: 전체 너비 (터치 스크롤)

## 3. 컴포넌트별 데이터 매핑

| Zone | Component | 데이터 소스 | 컬럼/로직 | 제약사항 |
|------|-----------|------------|----------|---------|
| Header | `UserNickname` | `USERS` | `U_NAME` | 로그인 필수 |
| Header | `ShareButton` | - | - | 클릭 시 공유 모달 |
| Hero | `DDayBadge` | `USERS` | `NOW() - U_REG_DATE` | D+일수 표시 |
| Metrics | `MetricCard_Post` | `BOARD_MUZZIMA` | `COUNT(*)` | `DEL_FLAG='N'` 조건 |
| Metrics | `MetricCard_Comment` | `COMMENT` | `COUNT(*)` | 인덱스(U_ID) 필수 |
| Metrics | `MetricCard_Like` | `BOARD_MUZZIMA` | `SUM(AGREE_CNT)` | `COALESCE(,0)` |
| Metrics | `MetricCard_Point` | `POINT_GRANT` | `SUM(POINT)` | 양수만 합계 |
| Chart | `TrendChart` | `BOARD_MUZZIMA`, `COMMENT` | `GROUP BY MONTH` | 최근 12개월 |
| Share | `ShareCard` | All metrics | 이미지 렌더링 | Canvas/html2canvas |

## 5. 인터랙션 정의

| 요소 | 이벤트 | 동작 |
|-----|-------|------|
| `ShareButton` | click | 공유 모달 오픈 |
| `MetricCard` | hover | 툴팁 표시 (상세 설명) |
| `ChartToggle` | click | 월별/주별 데이터 전환 |
| `DownloadButton` | click | 이미지 생성 후 다운로드 |
| `Chart` | touch/drag | 모바일에서 스크롤 |
