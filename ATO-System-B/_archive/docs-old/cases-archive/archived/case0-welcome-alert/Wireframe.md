# Wireframe.md - 환영 알림 화면 설계

## 레이아웃 구조
```
┌─────────────────────────────────────┐
│              App Container           │
│  ┌─────────────────────────────────┐ │
│  │     WelcomeNotification        │ │
│  │  ┌─────────────────────────┐   │ │
│  │  │  🎉 Welcome to          │ │ │
│  │  │     Medi-Notion!        │ │ │
│  │  │                    ✕    │ │ │
│  │  └─────────────────────────┘   │ │
│  └─────────────────────────────────┘ │
│                                     │
│         [Other App Content]         │
└─────────────────────────────────────┘
```

## 스타일 속성
- **위치**: Fixed, top-right (상단 우측 고정)
- **크기**: 300px × 80px
- **배경**: 그라데이션 (메디게이트 브랜드 컬러)
- **애니메이션**: Slide-in from right → Fade out

## 반응형 설계
- **Desktop**: 우측 상단 고정
- **Mobile**: 전체 너비, 상단 고정
- **Tablet**: Desktop과 동일

## 상호작용
- **자동 표시**: 3초 후 슬라이드인
- **자동 숨김**: 5초 후 페이드아웃
- **수동 닫기**: X 버튼 클릭 시 즉시 숨김
