#### 컴포넌트

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| Header | 공통 | 서비스명 + 날짜 |
| ScriptViewer | 핵심 | Host/Guest 대화 표시 |
| PlayControls | 핵심 | 재생/일시정지/진행바 |
| BestPostList | 보조 | 베스트 게시물 목록 |
| BestPostCard | 반복 | 개별 게시물 카드 |

#### 인터랙션

| 액션 | 이벤트 | 결과 |
|------|--------|------|
| Play 버튼 클릭 | onClick | 재생 시작/일시정지 토글 |
| 진행바 클릭 | onClick | 해당 위치로 이동 |
| 이전/다음 클릭 | onClick | 이전/다음 토픽으로 이동 |
| BestPostCard 클릭 | onClick | 해당 토픽으로 이동 |

### S002: ScriptViewer (대본 표시)

#### 레이아웃 (ASCII)

```
+------------------------------------------+
|  +--------------------------------------+ |
|  | [Host 아이콘]                        | |
|  | Host:                                | |
|  | 안녕하세요, 오늘의 무찌마 일간        | |
|  | 베스트입니다. 바쁜 선생님들을 위해    | |
|  | 3분 만에 핵심만 전달해 드릴게요.      | |
|  +--------------------------------------+ |
|                                          |
|  +--------------------------------------+ |
|  |                        [Guest 아이콘]| |
|  |                              Guest:  | |
|  |       네, 오늘도 흥미로운 이야기들이  | |
|  |       많네요. 첫 번째 주제부터        | |
|  |       살펴볼까요?                     | |
|  +--------------------------------------+ |
+------------------------------------------+
```

#### 컴포넌트

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| ScriptBubble | 반복 | 대화 말풍선 |
| SpeakerIcon | 장식 | Host/Guest 아이콘 |
| HighlightText | 강조 | 현재 재생 중인 텍스트 |

### S003: PlayControls (재생 컨트롤)

#### 레이아웃 (ASCII)

```
+------------------------------------------+
|                                          |
|      [◀◀]    [ ▶ / ⏸ ]    [▶▶]         |
|                                          |
|  ──────────●─────────────────────────── |
|  0:00                              3:00  |
|                                          |
+------------------------------------------+
```

#### 컴포넌트

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| PlayPauseButton | 핵심 | 재생/일시정지 토글 |
| SkipButton | 보조 | 이전/다음 토픽 |
| ProgressBar | 핵심 | 재생 진행률 표시 |
| TimeDisplay | 정보 | 현재/전체 시간 |
