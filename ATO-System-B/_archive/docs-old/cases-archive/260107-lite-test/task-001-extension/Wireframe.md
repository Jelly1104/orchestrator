#### 컴포넌트

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| Header | 공통 | 타이틀 + Refresh 버튼 |
| SkillCard | 반복 (x7) | 개별 Skill 정보 카드 |
| StatusBadge | 내장 | active/inactive 상태 표시 |
| Footer | 공통 | 통계 요약 |

#### 인터랙션

| 액션 | 이벤트 | 결과 |
|------|--------|------|
| Refresh 클릭 | onClick | 페이지 새로고침 |
| Card hover | onMouseEnter | 그림자 강조 효과 |
