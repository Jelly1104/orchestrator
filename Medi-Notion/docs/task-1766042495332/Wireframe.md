# Wireframe.md - 화면 설계

## 홈 페이지 레이아웃 (`/`)

### 헤더 영역
```
┌─────────────────────────────────────────────────┐
│                 Hello World App                 │
└─────────────────────────────────────────────────┘
```

### 메인 컨텐츠 영역
```
┌─────────────────────────────────────────────────┐
│                                                 │
│            🌍 Hello, World! 🌍                  │
│                                                 │
│     Welcome to our Hello World application     │
│                                                 │
│  ┌─────────────────┐  ┌─────────────────────┐   │
│  │   Test API      │  │    Response Area    │   │
│  │  [Call API]     │  │                     │   │
│  └─────────────────┘  │  (API response      │   │
│                       │   will appear here) │   │
│                       └─────────────────────┘   │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 푸터 영역
```

```

## 컴포넌트 명세

### 1. HelloWorldMessage 컴포넌트
- **위치**: 메인 중앙
- **기능**: 정적 Hello World 메시지 표시
- **스타일**: 대형 텍스트, 중앙 정렬

### 2. ApiTestButton 컴포넌트
- **위치**: 좌측 영역
- **기능**: API 호출 트리거
- **상태**: 로딩 중 비활성화

### 3. ResponseDisplay 컴포넌트
- **위치**: 우측 영역
- **기능**: API 응답 결과 표시
- **데이터 매핑**: `response.message`, `response.timestamp`

## 반응형 디자인
- **Mobile**: 세로 스택 레이아웃
- **Desktop**: 좌우 분할 레이아웃
