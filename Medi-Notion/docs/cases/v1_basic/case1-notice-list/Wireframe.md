## 2. 컴포넌트 데이터 매핑

| Zone | 컴포넌트 | 데이터 소스 | 비고 |
|------|---------|------------|------|
| Header | `PageTitle` | 하드코딩 | "공지사항" |
| List | `NoticeItem` | `BOARD_NOTICE` | 반복 렌더링 |
| Item.Badge | `Badge` | `NOTICE_TYPE` | '중요'/'일반' |
| Item.Title | `Text` | `TITLE` | 50자 truncate |
| Item.Date | `Text` | `REG_DATE` | YYYY-MM-DD |
| Item.Views | `Text` | `READ_CNT` | 숫자 포맷팅 |
| Pagination | `Pagination` | `totalCount / pageSize` | 10건 단위 |

## 4. 상세 컴포넌트 매핑

| Zone | 컴포넌트 | 데이터 소스 | 비고 |
|------|---------|------------|------|
| Header | `BackButton` | - | /notice로 이동 |
| Content.Title | `Heading` | `TITLE` | 전체 표시 |
| Content.Meta | `Text` | `REG_DATE`, `READ_CNT` | |
| Content.Body | `HTML` | `CONTENT` | XSS 방지 필수 |
| Footer | `Button` | - | /notice로 이동 |

## 5. 반응형 breakpoint

| 화면 | 너비 | 레이아웃 |
|------|------|---------|
| Desktop | ≥ 1024px | 좌우 여백 있음, 최대 800px |
| Tablet | 768-1023px | 좌우 여백 축소 |
| Mobile | < 768px | 풀 width, 패딩 16px |
