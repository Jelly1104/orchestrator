# G1 Use Case Trigger 후보 제안

> **작성일**: 2025-12-18
> **기반 데이터**: Case #4 활성 회원 패턴 분석 PoC
> **목표**: Agent 사용률 5% → 15%

---

## 1. 배경

### 1.1 현황 문제

| 항목 | 현황 | 문제점 |
|------|------|--------|
| Agent 사용률 | 5% 미만 | 홈페이지 팝업에도 대부분 닫기 클릭 |
| Use Case | 불명확 | "왜 써야 하는지" 사용자가 모름 |
| 타겟 세그먼트 | 미정의 | 어떤 회원에게 적용할지 근거 없음 |

### 1.2 접근 전략: 성공 복제 (Offensive)

**핵심 질문**: "플랫폼을 잘 쓰는 사람들에게 먼저 Agent를 적용하면?"

```
HEAVY 세그먼트 (19,868명, 15.5%)
       ↓
가장 활발한 사용자 = 새로운 기능 수용도 높음
       ↓
이 세그먼트 대상 Agent Use Case 우선 설계
       ↓
G1 목표 달성 가속 (5% → 15%)
```

---

## 2. HEAVY 세그먼트 프로파일

### 2.1 확인된 특성

| 지표 | 값 | 의미 |
|------|-----|------|
| 회원 수 | **19,868명** | 의사 회원의 15.5% |
| 월 평균 접속일 | **15.3일** | 2일에 1번 이상 접속 |
| 월 평균 세션 | **24.2회** | 활발한 플랫폼 이용 |
| 프로필 입력률 | **100%** | 정확한 타겟팅 가능 |

### 2.2 예상 프로파일 (추가 검증 필요)

```
┌─────────────────────────────────────────────────────────────┐
│  1차 타겟 세그먼트 (가설)                                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  전문과목: 내과 / 가정의학과 / 정형외과                          │
│  근무형태: 개원의 (전체 대비 과대 대표 예상)                      │
│  활동 패턴: 커뮤니티 중심 (정보 탐색형)                          │
│                                                             │
│  ✅ 1차 타겟: "개원 내과의"                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 3. Use Case Trigger 후보

### 3.1 Trigger 후보 1: 커뮤니티 인사이트 요약

**대상**: HEAVY 세그먼트 중 커뮤니티 활동 회원

```
┌──────────────────────────────────────────────────────────┐
│  Use Case: 커뮤니티 인사이트 요약                           │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  트리거: 사용자가 특정 게시판에 7일 연속 접속 시             │
│                                                          │
│  Agent 액션:                                              │
│  "이번 주 [무찌마] 게시판에서 가장 핫한 주제는              │
│   '00약 급여 확대' 입니다. 관련 게시글 3개를 요약해          │
│   드릴까요?"                                              │
│                                                          │
│  기대 효과:                                               │
│  - 시간 절약 가치 명확 (정보 탐색 → 요약 제공)              │
│  - 커뮤니티 활동 증가 (요약 → 원문 클릭)                    │
│  - Agent 가치 인지 향상                                    │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 3.2 Trigger 후보 2: 맞춤 채용 알림

**대상**: HEAVY 세그먼트 중 경력 관심 회원

```
┌──────────────────────────────────────────────────────────┐
│  Use Case: 맞춤 채용 알림                                  │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  트리거: 사용자 프로필(전문과목+지역) + 채용 신규 등록 시     │
│                                                          │
│  Agent 액션:                                              │
│  "[서울 강남] 지역에 [내과] 전문의 채용 공고가               │
│   3건 새로 등록되었습니다. 관심 있으신가요?"                 │
│                                                          │
│  기대 효과:                                               │
│  - 개인화 가치 체감 (프로필 기반 추천)                       │
│  - B2C 전환율 향상 (채용 서비스 유료화)                      │
│  - 프로필 완성도 독려                                       │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 3.3 Trigger 후보 3: 동료 의사 활동 알림

**대상**: HEAVY 세그먼트 전체

```
┌──────────────────────────────────────────────────────────┐
│  Use Case: 동료 의사 활동 알림                              │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  트리거: 같은 전문과목 동료 의사가 인기 게시글 작성 시        │
│                                                          │
│  Agent 액션:                                              │
│  "[내과] 동료 의사가 작성한 게시글이 100개 이상             │
│   공감을 받았습니다. 확인해 보시겠어요?"                     │
│                                                          │
│  기대 효과:                                               │
│  - 소셜 증거(Social Proof) 활용                           │
│  - 커뮤니티 참여 증가                                      │
│  - FOMO(Fear of Missing Out) 자극                        │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

## 4. 우선순위 및 실행 계획

### 4.1 우선순위 매트릭스

| 후보 | 구현 난이도 | 기대 효과 | 우선순위 |
|------|------------|----------|---------|
| 커뮤니티 인사이트 요약 | 중 | 높음 | **1순위** |
| 맞춤 채용 알림 | 낮음 | 중간 | 2순위 |
| 동료 의사 활동 알림 | 낮음 | 중간 | 3순위 |

### 4.2 1순위 실행 계획

**커뮤니티 인사이트 요약** 상세 설계:

```
Phase 1: MVP (2주)
├── 대상: HEAVY 세그먼트 중 무찌마 게시판 활동 회원
├── 트리거: 7일 연속 접속 + 10개 이상 게시글 조회
├── 기능: 주간 인기 게시글 3개 요약 제공
└── 측정: Agent 클릭률, 요약 후 원문 클릭률

Phase 2: 확장 (2주)
├── 대상: HEAVY 세그먼트 전체
├── 트리거: 게시판별 맞춤 설정
├── 기능: 개인화 요약 (관심 주제 기반)
└── 측정: Agent 재사용률, DAU 변화
```
